# Student Management API

## Features

- Create Student
- Get All Students
- Get Student By ID
- Update Student
- Delete Student
- Partial Update (PATCH)

## Tech Stack

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Lombok
- ModelMapper
- Swagger

## API Documentation

Uncomment Swagger dependency in pom.xml

Then open:

http://localhost:8080/swagger-ui/index.html (Uncomment the swagger dependecy)
