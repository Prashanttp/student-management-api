# Student Management System

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Lombok
- Swagger/OpenAPI
- HTML/CSS/JavaScript Frontend

## Features
- Add Student
- View Student
- Update Student
- Delete Student
- Search Student by ID
- Interactive Dashboard

## API Documentation
Uncomment Swagger dependency and run:
http://localhost:8080/swagger-ui/index.html
